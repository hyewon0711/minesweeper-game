공간 ① 아늑한 서재 — 팝아트 / 카툰 스타일 시안

기존 픽셀-아트(<rect> 기반) 그래픽을 카툰 일러스트 스타일로 다시 그려본 시안이에요. 스테이지마다 디테일이 추가되는 reveal 시스템(data-min-stage)은 그대로 호환됩니다. 스타일 키워드: 따뜻한 크림 벽 + 우드 플로어, 굵은 라인, 평면 컬러, 카툰 비율.

1. 기존 스타일 vs 새 스타일 (둘 다 reveal level 10 = 완성본)

기존 — 픽셀 아트 (어두운 야간 톤)
새 시안 — 카툰 / 팝아트 (따뜻한 주간 톤)

2. 새 시안의 reveal 단계 (스테이지 1 → 10)

왼쪽 → 오른쪽, 위 → 아래로 갈수록 디테일이 쌓입니다.

3. 슬라이더로 직접 단계 변경

스테이지 10

4. 스테이지별로 추가되는 요소

1크림 벽 + 우드 플로어 (베이스 룸)
2중앙 안락 의자 (머스타드 + 세이지 쿠션)
3아치형 창문 + 커튼, 햇빛 자국
4왼쪽 책장 (다섯 단, 컬러 책)
5플로어 램프 (따뜻한 빛 글로우)
6사이드 테이블 + 펼친 책
7안락 의자 위 잠든 고양이
8구석 화분 (몬스테라)
9벽 액자 두 점 (추상 + 풍경)
10김 나는 찻잔, 벽시계, 책 더미
