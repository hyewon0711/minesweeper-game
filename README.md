# 지뢰찾기 (Minesweeper)

스테이지 기반 지뢰찾기 웹 게임. 순수 HTML/CSS/JS로 구현.

## 실행

`index.html`을 브라우저로 열면 됩니다. (별도 빌드 과정 없음)

## 파일 구조

- `index.html` — 게임 화면 마크업과 모달(게임오버/클리어/엔딩)
- `style.css` — 보드/셀/모달 스타일
- `script.js` — 게임 로직과 스테이지 진행

## 다른 컴퓨터에서 개발하기

```bash
git clone <이 저장소 URL>
cd 지뢰찾기
# Claude Code 실행
claude
```
